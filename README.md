# Classification of Oil and Gas Fields

## Описание

Проект посвящён задаче многоклассовой классификации нефтегазовых месторождений по типу размещения: onshore, offshore и смешанный тип. Решение выполнено в рамках соревнования Kaggle с использованием методов машинного обучения и ориентировано на метрику F1-macro с учётом дисбаланса классов.

## Задача и данные

По совокупности географических, геологических и технологических признаков необходимо определить тип месторождения. Используются обучающая и тестовая выборки с числовыми и категориальными признаками; целевая переменная — `Onshore/Offshore` (0, 1, 2).

## Подход

Выполнен полный цикл анализа: EDA, предобработка пропусков, feature engineering (net_to_gross, log_depth), построение пайплайна `ColumnTransformer + Pipeline`, сравнение нескольких моделей и подбор гиперпараметров. В качестве основной модели выбран Gradient Boosting.

## Результаты

Лучшая модель достигает F1-macro около 0.63 на кросс-валидации и accuracy около 0.85 на валидации. Наиболее значимыми оказались географические признаки, координаты и физические характеристики коллектора. Сформирован файл `submission.csv` для загрузки на Kaggle.
